# 传统监督学习 - 要点梳理

## 1. 监督学习基本问题


* **分类问题**：核心在于预测非连续的、离散的数据 。模型的输出通常为类别标签，如 $\boldsymbol{Y} \in \{0,1\}$ 或 $\boldsymbol{Y} \in \{1,2,\dots,k\}$ 。常用的误差函数形式为分类错误指示函数的求和：$L = \sum_{i=1}^n \mathbb{I}(f(x_i) \neq y_i)$ 。



* **回归问题**：核心在于预测连续的、具体的数据 。模型的输出通常为实数集中的某个值，即 $\boldsymbol{Y} \in \mathbb{R}$ 。常用的误差函数形式为预测值与真实值差值的平方和：$L = \sum_{i=1}^n (f(x_i) - y_i)^2$ 。



## 2. 线性模型与最小二乘法 (回归)

* **核心思想**：通过最小化预测值与真实值的平方误差来进行拟合，适用于特征与目标呈线性关系的连续值预测 。


* **优缺点比较**：优点在于简单、可解释性强且计算速度快 。缺点是对异常值较为敏感，并且严格要求数据符合线性假设 。


* **矩阵解析解**：对于线性关系 $\boldsymbol{F} = \boldsymbol{X}\boldsymbol{K}$，在最小化误差函数后，可以得到参数的矩阵形式解：$\boldsymbol{K} = (\boldsymbol{X}^T\boldsymbol{X})^{-1}\boldsymbol{X}^T\boldsymbol{F}$ 。


* **几何意义**：对于方程 $\boldsymbol{A}\boldsymbol{x} \approx \boldsymbol{b}$，最小二乘法的几何意义是在 $\boldsymbol{A}$ 的列空间中寻找一个拟合向量 $\boldsymbol{A}\boldsymbol{x}$，使得其与高维目标向量 $\boldsymbol{b}$ 之间的距离最短 。该问题在欧式距离下存在唯一的最小解 。



## 3. 线性分类模型

* **逻辑回归 (LR)**：
* **原理**：虽然名为回归，但实际是分类模型，它通过 Sigmoid 函数将线性组合映射为 $0$ 到 $1$ 之间的概率输出 。


* **误差函数**：相较于不连续的 $0/1$ 分类误差函数，逻辑回归采用具有概率解释的对数误差函数：$L = -\frac{1}{n} \sum_{i=1}^n [y_i \ln f(\boldsymbol{x}_i) + (1-y_i) \ln(1-f(\boldsymbol{x}_i))]$ 。其连续性有利于基于梯度的优化算法 。




* **线性判别分析 (LDA)**：
* **核心假设**：假设各类数据服从相同协方差的正态分布 。


* **核心思想**：寻找一个投影方向，使样本在直线上的投影点按照不同类型聚集 。


* **优化目标**：最大化目标函数 $J$，即在减小同一类样本距离的同时，增大不同类样本间的距离 。



$$J = \frac{(\boldsymbol{w}^T\boldsymbol{\mu}_0 - \boldsymbol{w}^T\boldsymbol{\mu}_1)^2}{\boldsymbol{w}^T\boldsymbol{\Sigma}_0\boldsymbol{w} + \boldsymbol{w}^T\boldsymbol{\Sigma}_1\boldsymbol{w}}$$




* **多分类策略**：
* **OvO (One vs One)**：将 $N$ 个类别拆解为 $C_N^2$ 个二分类任务 。该方法耗时较长，但分类更准确且没有类别混淆 。


* **OvR (One vs Rest)**：将 $N$ 个类别拆解为 $N$ 个二分类任务 。这种方法更为常用，但容易引发类别不平衡的问题 。





## 4. 决策树

* **树形结构与生成**：决策树是一种非参数的非线性模型，其生成是一个递归过程 。算法通过不断递归地选择特征进行划分，对各个部分生成决策子树，直至分类结束 。


* **剪枝防止过拟合**：过大且过密的决策树很容易对训练数据敏感并导致“过拟合”现象，因此必须进行“剪枝”处理以保障其泛化能力 。


* **单变量 vs 多变量**：
* 变量决策树的分类边界必定垂直于坐标轴，遇到复杂的真实边界时，需要采用大量分段来进行近似 。


* 变量决策树的分类边界未必垂直于坐标轴，具有更强的拟合真实边界的能力，但由于自由度过高，也会使得学习过程变得困难 。





## 5. 支持向量机 (SVM)

* **分类间隔与超平面**：SVM 的目标是在样本空间中寻找到一个使间隔最大的超平面 $\boldsymbol{w}^T\boldsymbol{x} + b = 0$ 。所有样本点应尽量远离分类超平面 。


* **支持向量 (Support Vector)**：在优化过程中，使得方程 $\boldsymbol{w}^T\boldsymbol{x} + b = \pm 1$ 等号成立的样本被称为“支持向量” 。两个异类支持向量到超平面的距离之和称为“间隔”，其大小等于 $\frac{2}{\|\boldsymbol{w}\|}$ 。


* **硬间隔与软间隔优化**：
* 间隔要求所有样本必须严格满足分类条件 $y_i(\boldsymbol{w}^T\boldsymbol{x}_i + b) \ge 1$ 。


* 间隔通过引入非负的松弛变量 $\xi_i \ge 0$，允许个别样本存在误差，其优化目标演变为求解：最大化 $\frac{2}{\|\boldsymbol{w}\|} - C\sum_{i=1}^m \xi_i$ 。




* **核函数技巧 (Kernel Trick)**：支持向量机本质上是线性模型，但通过核函数的映射，可以将低维度下线性不可分的数据映射到更高维度，从而在特征空间中转化为线性可分问题 。
