# 传统机器学习：无监督学习 - 要点梳理

## 1. 核心概念与任务

* **无监督学习目标**：挖掘数据中隐含的属性 。


* **主要分析任务**：包括按照属性对数据进行分类的聚类任务、寻找最关键属性的降维分析、估计属性概率密度分布的密度估计、根据属性判断异常的异常检测，以及找到属性间关系的关联学习 。



## 2. 聚类学习 (Clustering)

* **聚类问题定义**：寻找数据集中的互不相交的子集，使每个子集自动形成“簇”结构，可作为其余分析的预处理 。核心原则是寻找“簇内”相似性与“簇间”差异性，即类内距离尽可能小，类间距离尽可能大 。


* **聚类效果定量刻画指标**：
* **簇内指标**：类内平均距离、类内样本分布直径 。


* **簇间指标**：类间最小距离、类间中心距离 。




* **距离的代数定义与性质**：代数中距离需满足非负性、同一性、对称性和直递性（三角不等式） 。


* **常用距离计算**：最常用的是闵可夫斯基距离（Minkowski distance） 。其中 p=1 时为曼哈顿距离（街区距离），p=2 时为欧氏距离 。



## 3. 代表性聚类算法

* **K-means 聚类**：
* **算法原理**：基于初始 k 个代表性样本的优化，采用贪心算法策略，使得每个样本点归属于距离最近的簇中心 。


* **优化目标**：使得每一簇内样本尽可能聚拢，即求解均方误差极小值的最优解问题 。     
* **主要局限**：无法动态调整簇个数（k） 。




* **高斯混合聚类 (Mixture-of-Gaussian)**：
* **算法原理**：假设样本的每个簇服从高斯分布，使用多个高斯分布的和来进行估计整体样本的分布 。


* **求解步骤**：选定初始参数后，计算所有样本由各混合成分生成的后验概率作为分簇依据，随后根据样本后验分布通过极大似然估计计算新的均值、协方差矩阵和混合系数，直至参数不再变化（收敛） 。


* **主要局限**：同样无法动态调整簇个数（k） 。





## 4. 数据降维与维数灾难 (Dimensionality Reduction)

* **高维数据问题**：当维度 $d$ 较大时即为高维数据，例如围棋棋盘状态、文本的 one-hot 向量、包含像素值的高维图像数据、以及神经元活动记录等 。高维数据会导致样本稀疏、距离计算困难、采样不充分等“维数灾难”问题 。


* **降维本质**：寻找高维空间的低维嵌入（如将三维空间映射到二维曲面），在低维子空间中尽可能保留样本间关联的更多信息 。



## 5. 典型降维算法对比

* **主成分分析 (PCA) - 线性降维**：
* **原理**：最常用的数据降维方法，旨在寻找最佳低维子空间以保留主要信息 。其核心是通过对协方差矩阵 $\mathbf{C} = \mathbf{X}\mathbf{X}^\top$ 进行特征值分析来实现降维 。


* **局限**：作为线性降维方法，它无法准确刻画三维空间中S型曲面等结构的非线性关联，此时需要引入非线性降维方法 。




* **多维放缩 (MDS) - 基于距离降维**：
* **原理**：核心目标是使得高维空间中样本间的距离与降维后低维空间中样本间的欧氏距离保持一致 。




* **等度量映射 (ISOMAP) - 非线性降维**：
* **原理**：对于复杂结构，三维空间的欧氏距离不再合适；ISOMAP 的核心想法是运用测地线距离（沿曲面距离）代替欧氏距离，随后根据曲面上的距离进行降维刻画 。 

* **随机近邻嵌入 (t-SNE) - 非线性降维**：


* **原理**：通过概率密度分布定义距离，目标是使得高维空间与低维空间中的条件概率分布（衡量样本相似度）尽可能相似 。


* **距离分布假设**：高维空间中样本间的距离采用高斯分布建模，而降维空间中样本间的距离采用学生t分布建模 。


* **优化指标**：使用 Kullback-Leibler (KL) 散度来衡量并最小化这两个概率分布之间的距离 。





## 6. 经典应用实践

* **主成分分析 (PCA) 应用**：可用于图像降噪，例如对带噪声的 MNIST 数据集进行处理时，可通过保留前50个主成分来实现数据的去噪重构 。


* **K-means 算法应用**：可应用于处理西瓜数据集，通过密度和含糖率等属性对无标记的样本进行分类运算 。
